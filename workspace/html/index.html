퍼블리싱용 Gulp 세팅

사용 모듈
clean : 배포 폴더 클린
sass : CSS 전처리기
gulpEjs : 템플릿 언어
sprite : 이미지 스프라이트
html : html 복사
script:build : 스크립트 병합, 압축, 이름 변경
imagemin : 이미지 최적화
library : 라이브러리 복사
fonts : 폰트 복사
nodemon : 서버 구동
browserSync : 라이브 로딩
watch : 파일 변경 체크
