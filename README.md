# Gulp-Publishing (퍼블리싱을 위한 Gulp 세팅)

## 1. Gulp-cli  & Gulp설치

```
npm install gulp-cli -g
npm init
npm install gulp --save-dev
```

## 2. Sass 설치

```
npm install gulp-sass --save-dev
npm install gulp-sourcemaps --save-dev
```
